Regular Expression - RE is a special sequence of character that uses a search pattern to find a string or set of string. It can detect the presence or absence of text by matching it with a particular pattern, and also can split a pattern into one or more sub patterns.

Import re

Python provides a re module whose primary function is to offer a search, where it takes a regular expression and a string. Uses - Find matching pattern in a string, phone verification, web scrapping.
RegEx Module Cheetsheet

Functions of RegEx


1. re.findall()-

Returns all non overlapping matches of pattern in string, as a list of string the string is scanned left to right and matches are returned in the order found.

2. re.compile()-

Regular expressions are compiled into pattern objects, which have methods for various operations such as searching for pattern matches or performing string substitutions



3. re.split()-

Split string by the occurance of character or a pattern, when finding that pattern, the remaining characters from the string are returned as part of the resulting list.

3. re.sub()-

The method returns a string where matched occurances are replaced with the content of replace variable.

4. re.subn()-

sub() is similiar to sub() in all ways, except in its way of providing output, it returns a tuple with a count of the total of replacement and new string rather than just the string.

5. re.escape()-

Returns string with all non-alphanumeric backslash, this is useful if you want to match an arbitrary literal string that may have regular expression mata characters in it.

6. re.search()-

Method either returns none(if pattern doesn't match), this method stops after the first match, so this is best suited for testing a regular expression more than extracting data.

7. re.match()-

re.match() function of re in Python will search the regular expression pattern and return the first occurrence. The Python RegEx Match method checks for a match only at the beginning of the string. So, if a match is found in the first line, it returns the match object.
